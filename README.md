# mini-socket-server-and-client-for-chat

This is a simple socket server and client to chat on port 53

# ⚠️ Warning ⚠️

ONLY RUN ON LINUX

one talk to the other, after its reverse like client -> server, server -> client

You can only make word ex: how are you -> how_are_you

# Compile it

Using GCC

```bash
gcc client.c -o client
```

Using Clang


```bash
clang client.c -o client
```


Its the same for server.c
